# desafio-felipao-rankeadas
código javascript para jogadores em  partidas rankeadas 
Leva em consideração a quantidade de vitórias e derrotas
Dependendo da pontuação o jogador vai receber um nível
pontos = vitórias -derrotas
Nível do jogador conforme escala:

Se quantidade de vitórias menor do que 10 - Nível Ferro
Se quantidade de vitórias entre 11 e 20  - Nível Bronze
Se quantidade de vitórias entre 21 e 50  - Nível Prata
Se quantidade de vitórias entre 51 e 80  - Nível Ouro
Se quantidade de vitórias entre 81 e 90  - Nível Diamante
Se quantidade de vitórias entre 91 e 100  - Lendário
Se quantidade de vitórias maior ou igual a 101  - Imortal
